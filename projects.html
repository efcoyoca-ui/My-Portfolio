CSAM 112 FINAL PROJECT | SHOWCASING HOW PYTHON LANGUAGE TURN INTO 3D FACEBOOK LOGO MODEL

This project demonstrates 3D geometric transformations applied to a custom 3D model of the Facebook “f” logo using Python. The program visualizes how translation, rotation, scaling, and reflection affect the object in 3D space. It uses Matplotlib for rendering and NumPy for performing matrix-based transformations. The project highlights how linear algebra operations can manipulate 3D graphics efficiently.
